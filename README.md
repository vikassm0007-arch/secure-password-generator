# 🔐 Random Password Generator (Java)

A simple yet powerful Java-based Random Password Generator that allows users to create secure passwords based on customizable criteria.

## 🚀 Features

- Generate passwords of any desired length
- Option to include:
  - Lowercase letters (a-z)
  - Uppercase letters (A-Z)
  - Numbers (0-9)
  - Special characters (!@#$%^&*...)
- Ensures randomness using Java's built-in `Random` class
- User-friendly console interaction

## 🛠️ Technologies Used

- Java
- Scanner (for user input)
- Random (for password generation)
- StringBuilder (for efficient string handling)

## 📋 How It Works

1. User enters the desired password length
2. Selects character types to include
3. Program builds a character pool dynamically
4. Random characters are selected to generate the password
5. Final password is displayed
